1번. 선택정렬을 위한 슈도 코드를 완성하시오.

selection_sort(N,arr):
	for i in range(N-1):
		min_idx = i
		for j in range(1+i,N):
			if arr[j]<arr[min_idx]:
				min_idx = j
		arr[i],arr[min_idx] = arr[min_idx],arr[i]
	return arr
